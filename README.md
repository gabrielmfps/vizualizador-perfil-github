# 🚀 Visualizador de Perfil do GitHub

Este é um projeto simples e funcional que utiliza a API do GitHub para buscar informações de perfis de usuários e listar seus repositórios. Desenvolvido com foco no aprendizado de JavaScript Intermediário (Consumo de APIs, Promises e Manipulação de DOM).

## ✨ Funcionalidades

- 🔍 **Busca de Usuários**: Encontre qualquer perfil público do GitHub pelo nome de usuário.
- ⌨️ **Atalho de Teclado**: Realize buscas pressionando apenas a tecla **Enter**.
- 👤 **Informações do Perfil**: Exibe foto de avatar, nome, biografia, seguidores e quem o usuário está seguindo.
- 📂 **Repositórios**: Lista os repositórios públicos do usuário.
- 📱 **Responsivo**: Layout adaptável para dispositivos móveis e desktops.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica.
- **CSS3**: Estilização com variáveis, Grid, Flexbox e animações.
- **JavaScript (ES6+)**: 
  - Módulos (ES Modules) para organização de código.
  - `fetch` API para comunicação com o GitHub.
  - Manipulação de eventos e DOM.
- **API do GitHub**: Fonte de dados dos perfis.

1. Clone este repositório:
   ```bash

   ```
